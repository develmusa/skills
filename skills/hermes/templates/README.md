# HERMES Templates

Markdown templates for every HERMES deliverable and checklist, converted from the official HERMES 2022 templates. Each file keeps the official section structure and fill-in guidance (shown in quote blocks). Replace every `[ ... ]` placeholder and delete the guidance lines and unused rows before use.

To produce a deliverable, open the matching template here and consult [../references/deliverables.md](../references/deliverables.md) for what the document is for and [../references/tasks.md](../references/tasks.md) for who produces it and when.
