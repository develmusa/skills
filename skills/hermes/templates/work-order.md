### Aim and purpose of the document

The work order contains all relevant information for completing a set task. The project management uses it to give orders to the project staff within the framework of project planning, management, information, and monitoring. Work orders can be issued internally or externally. Any solution-specific orders must be coordinated with the user representative in advance.
